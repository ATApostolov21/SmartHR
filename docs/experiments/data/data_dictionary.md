# Речник на данните (Data Dictionary)

Този документ описва структурата и полетата в набора от данни `Synthetic_Complex_Data.csv`, използван за анализ и моделиране на текучеството на персонал (Churn Prediction).

## Основна информация
*   **Файл:** `dataset/Synthetic_Complex_Data.csv`
*   **Предназначение:** Съдържа данни за служителите, тяхното представяне, удовлетвореност и статус на напускане.

## Описание на променливите (Fields)

### Идентификация и Демография
| Поле | Тип | Описание | Примерни стойности |
| :--- | :--- | :--- | :--- |
| **Employee_ID** | Integer | Уникален идентификатор на служителя. | 1, 2, 1005 |
| **Department** | String | Отделът, в който работи служителят. | IT, Finance, HR, Sales |
| **Gender** | String | Пол на служителя. | Male, Female |
| **Age** | Integer | Възраст на служителя в години. | 25, 40, 55 |
| **Job_Title** | String | Заемана длъжност. | Manager, Developer, Analyst |
| **Hire_Date** | DateTime | Дата и час на назначаване. | 2022-01-19 |
| **Years_At_Company** | Integer | Брой години стаж в текущата компания. | 2, 5, 10 |
| **Education_Level** | String | Образователна степен. | Bachelor, Master, PhD |

### Работни показатели и Компенсация
| Поле | Тип | Описание | Примерни стойности |
| :--- | :--- | :--- | :--- |
| **Performance_Score** | Integer | Годишна оценка на представянето (скала 1-5). | 1 (Low), 5 (High) |
| **Monthly_Salary** | Float | Месечна работна заплата (в основна валута). | 5000.0, 7500.0 |
| **Work_Hours_Per_Week**| Integer | Среден брой работни часове на седмица. | 40, 45, 50 |
| **Overtime_Hours** | Integer | Брой часове извънреден труд на месец (или период). | 0, 10, 25 |
| **Projects_Handled** | Integer | Брой проекти, по които служителят е работил. | 5, 12, 30 |
| **Promotions** | Integer | Брой получени повишения по време на престоя във фирмата.| 0, 1, 2 |

### Условия на труд и Удовлетвореност
| Поле | Тип | Описание | Примерни стойности |
| :--- | :--- | :--- | :--- |
| **Employee_Satisfaction_Score** | Float | Индекс на удовлетвореност на служителя (скала 1-5). | 2.5, 4.8 |
| **Remote_Work_Frequency** | Integer | Процент от времето, прекарано в дистанционна работа. | 0% (Office), 100% (Remote) |
| **Team_Size** | Integer | Брой хора в прекия екип на служителя. | 5, 15 |
| **Training_Hours** | Integer | Часове, прекарани в обучение и квалификация. | 20, 40 |
| **Sick_Days** | Integer | Брой дни болничен за последната година. | 2, 10 |

### Целева променлива (Target)
| Поле | Тип | Описание | Стойности |
| :--- | :--- | :--- | :--- |
| **Resigned** | Boolean | Дали служителят е напуснал компанията. Това е променливата, която моделите се опитват да предскажат. | `True` (Напуснал), `False` (Останал) |
